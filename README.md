# Invezt Application

Invezt is a full-stack web application designed to help users analyze, compare, and manage stock market investments efficiently. The platform provides intelligent insights, real-time data visualization, and financial tools to support better investment decisions.

This project was developed as part of the **Software Development Group Project (SDGP)** at the **University of Westminster**.

---

## Features

- **Interactive Dashboard** – Company report valuation and market overview
- **Stock Analysis** – 10 key financial ratios with star ratings (P/E, ROE, ROA, Debt-to-Equity, Current Ratio, Quick Ratio, EPS, Dividend Yield, P/B Ratio, PEG Ratio)
- **Company Comparison** – Compare up to 3 companies side-by-side with investment recommendations
- **Valuation Models** – DCF, CAPM, and Gordon Growth calculators
- **Portfolio Management** – Track holdings with real-time profit/loss
- **Watchlist** – Monitor favorite stocks with price alerts
- **Real-time Data** – Live CSE data updates every 60 seconds
- **Market News** – Latest CSE news with category filters
- **Learning Module** – Educational resources for beginner investors


## Technology Stack

**Frontend**

- React (Vite)
- HTML, CSS, JavaScript
- Recharts (Data Visualization)

**Backend**

- Node.js / Express.js
- REST APIs
- JWT Authentication

**Database**

- MongoDB

---

## Project Structure

- `frontend/` – React-based user interface
- `backend/` – Server-side logic and APIs
- `components/` – Reusable UI components
- `pages/` – Application pages (Dashboard, Analysis, etc.)

## Setup Instructions

### 1. Clone the Repository

git clone https://github.com/Pasandul-pro/invezt-application.git  
cd invezt-application

### 2. Install Dependencies

**Frontend**  
cd frontend  
npm install  
npm run dev

**Backend**  
cd backend  
npm install  
npm start
