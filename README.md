# Invezt Application

Invezt is a full-stack web application designed to help users analyze, compare, and manage stock market investments efficiently. The platform provides intelligent insights, real-time data visualization, and financial tools to support better investment decisions.

This project was developed as part of the **Software Development Group Project (SDGP)** at the **University of Westminster**.

---

## Features

- Interactive dashboard with company report valuation
- Stock analysis based on financial ratios
- Company comparison with AI-powered insights
- Portfolio and watchlist with real-time values and growth-over-time charts
- Valuation model references for better decision-making
- Stock market news integration
- Learning module with stock market resources

---

## Technology Stack

**Frontend**

- React (Vite)
- HTML, CSS, JavaScript
- Recharts (Data Visualization)

**Backend**

- Node.js / Express.js
- REST APIs
- JWT Authentication

**Database**

- MongoDB

---

## Project Structure

- `frontend/` – React-based user interface
- `backend/` – Server-side logic and APIs
- `components/` – Reusable UI components
- `pages/` – Application pages (Dashboard, Analysis, etc.)

## Setup Instructions

### 1. Clone the Repository

git clone https://github.com/Pasandul-pro/invezt-application.git  
cd invezt-application

### 2. Install Dependencies

**Frontend**  
cd frontend  
npm install  
npm run dev

**Backend**  
cd backend  
npm install  
npm start
